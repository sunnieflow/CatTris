## ✨ Features

- 🐱 Cat emoji particle effects on line clear
- 😿 Cat-themed game over screen
- 🎉 Level up burst animation
- 🔊 Satisfying bubble pop sound effects (Web Audio API)
- 👻 Ghost piece preview
- 📱 Mobile touch support
- 🌐 Zero dependencies — single HTML file

---

## 🕹️ Controls

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Move left / right | `← →` | Swipe left / right |
| Rotate | `↑` | Tap ↑ button |
| Soft drop | `↓` | Tap ↓ button |
| Hard drop | `Space` | Tap DROP / touchend |

## 🚀 Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/cattris.git
cd cattris
open cattris.html   # macOS
# or just double-click the file
```

No build step. No npm install. No framework. Just one file.

---

## 🌍 GitHub Pages Deploy

1. Go to repo **Settings → Pages**
2. Source: `main` branch, `/ (root)`
3. Save → your game is live at `https://YOUR_USERNAME.github.io/cattris`
